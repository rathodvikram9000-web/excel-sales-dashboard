# Sales Performance & Customer Insights Dashboard (Excel)

## Domain
Retail / E-Commerce

## Objective
Analyze sales data to understand revenue trends, profitability, and customer behavior.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Timelines

## Key Features
- KPI Cards (Sales, Profit, Profit Margin, AOV, Customers)
- Sales Trend Analysis
- Region-wise Sales Performance
- Category-wise Profit Analysis
- Top 10 Products by Sales
- Interactive Slicers
